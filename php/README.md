# mat_kul php

Aplikasi CRUD berbasis PHP dan MySQL.

## Instalasi

1. Ubah folder `php` ke `matkul` taruh di folder `/htdocs` dan jalankan XAMPP
2. Import file `matkul.sql` lewat PHPMyAdmin

## Database

Database Name: matkul
Database Username: root
